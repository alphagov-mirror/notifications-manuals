Dbeaver

This document describes how to connect to the local database using Dbeaver, and create a read-only connection to the produciton database.

Local database connection

  1. Create a new connection. Use these configurations
  2. The tables are visible under Schemas/public/Tables

Create a read-only conneciton to database on server

  1. You need to establish a database connection from shell. Following the procedure in this documentation, you create an SSH tunnel to our server - PaaS Note
  2. From Dbeaver, make a new database connection
  3. Select postgres as connection type
  4. Enter the parameters and credentails for this connection. The port should be 15432 as the SSH tunnel you created in step 1. The database name and credentials can be found in Notify's credentail in /credentail/pass/postgres_logins.txt
  5. Click Next
